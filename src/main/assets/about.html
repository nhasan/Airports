This application makes use of the data from following sources:

A/FD — Airport/Facility data is provided by FADDS - Facility Aeronautical Data Distribution System, part of the NFDC - National Flight Data Center. This data is updated on a 56-day cycle.

Timezone information is populated in the data file using GeoNames.

Privacy

This app uses Google Analytics to collect anonymous usage data. This data is solely intended to provide the developer with information to help improve the app. No personal information is associated with the data collected by Google Analytics.
